## ‚ú® Features

*   **Dynamic Card Battles**: Fast-paced 1vs1 combat with elemental advantages (Fire > Earth > Air > Water).
*   **Deck Building**: Draft your hand from an infinite deck of random cards before every match.
*   **Progression System**: Win battles to level up, refill your hand, and survive as long as possible.
*   **Card Album**: Collect unique cards and view your collection.
*   **Ranking**: Track your "Games Played", "Total Power", and "Score" directly from the main menu.
*   **Multiplayer**: Host or join local network games to play against friends.
*   **Localization**: Fully localized in English, Spanish, and Catalan.
*   **Cross-Platform**: Optimized for Mobile (vertical layout) and runs on Linux (custom window size).

## üöÄ Getting Started

This project is built with [Flutter](https://flutter.dev).

### Prerequisites
*   Flutter SDK
*   Dart SDK

### Run Locally

```bash
# Get dependencies
flutter pub get

# Run on Android/iOS (Mobile)
flutter run

# Run on Linux (Desktop)
flutter run -d linux
```

## üéÆ How to Play

1.  **Select Deck**: Swipe DOWN to pick a card for your hand (max 3), Swipe LEFT/RIGHT to discard.
2.  **Battle**: Drag your card into the arena to play it. The card with higher power wins! (Watch out for elemental bonuses +30).
3.  **Win**: If your score is higher at the end of the round, you win!
4.  **Collect**: Winners get to open a card pack to expand their album.
